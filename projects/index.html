Nuke Heavy Node Optimizer (Shipped)

One action to toggle “heavy” node classes across a script for faster iteration. Safe by design (only touches each node’s disable knob), configurable via presets, and suitable for shared studio deployment.

Nuke Python PySide2/6 Status: Shipped
  • Estimate: reduces preview-mode switching from ~20–60s of hunting/toggling nodes to ~1–2s (single action).
  • Safety: toggles only the node disable knob (no rewiring, reversible).
  • Config: JSON-backed presets (studio-friendly defaults + personal overrides).
  • Compatibility: Qt shim to support the PySide2 → PySide6 era (Nuke 13–16+).
