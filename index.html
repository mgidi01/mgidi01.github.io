Nuke Heavy Node Optimizer (Shipped)

When a comp gets heavy (denoise/defocus/retime/motion blur, etc.), I lose time repeatedly disabling/enabling nodes just to iterate. This Optimizer bulk-toggles a configurable list of heavy node classes across the script (it only touches the disable knob) + a tiny editor UI for the class list.

Tested: Windows — Nuke 13 / 15 / 16. PySide2 & PySide6.

Nuke Python PySide2/6 Status: Shipped
  • Ctrl+Alt+O to toggle your “heavy” node types
  • Edit the class list in a small UI
