Nuke Heavy Node Optimizer (Shipped)

When a comp gets heavy (Kronos, OFlow, Denoise…), I don’t want to toggle nodes one by one just to preview. This gives me one hotkey to flip a configurable set of node classes. It only touches each node’s disable knob, so it stays predictable.

Tested: Windows — Nuke 13 / 15 / 16. PySide2 & PySide6.

Nuke Python PySide2/6 Status: Shipped
  • One hotkey: working ↔ final (for the classes you choose).
  • Presets saved to JSON (easy to share in a studio).
  • UI works across PySide2/PySide6.

Nuke Toolsets Library (In progress)

A simple browser to find toolsets quickly and drop them into your script without digging through folders. It supports shared and personal libraries, and it’s being built to be loud when something is broken (warnings + how to fix).

In progress: make broken toolsets obvious + easy to fix (so artists aren’t blocked).
